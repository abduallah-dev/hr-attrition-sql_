# HR Employee Attrition Analysis

## Problem Statement
Company ke employee data mein SQL queries laga kar pata
kiya ki kaun se employees job chhod rahe hain, kyun,
aur kaunse department mein attrition sabse zyada hai.

## Tools Used
Python | Pandas | SQLite | Matplotlib

## Dataset
IBM HR Analytics Dataset — 1470 rows, 35 columns

## Key Findings
- Sales department mein attrition sabse zyada hai
- Low income employees zyada job chodthe hain
- Overtime karne wale employees 3x zyada leave karte hain

## Files
- `hr_analysis.py` — main analysis script
- `queries.sql` — SQL queries
- `charts/` — visualizations
